# Pin Mapping

| Signal | MCU Pin | Description |
|---|---|---|
| LCD Data D0–D7 | P1.0–P1.7 | 8-bit data bus |
| LCD RS | P2.0 | Register Select |
| LCD Enable | P2.2 | Strobe to latch data |
| LCD RW | GND | Write-only |
| Keypad Row 1 | P2.1 | Scan keys 1,2,3 |
| Keypad Row 2 | P2.3 | Scan keys 4,5,6 |
| Keypad Row 3 | P2.4 | Scan keys 7,8,9 |
| Keypad Col 1-3 | P2.5–P2.7 | Column detect |
| Mode Button | P3.7 | Toggle game mode |
